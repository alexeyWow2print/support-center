# Ширина рулона

В разделе Ширина рулона вы можете установить разные варианты ширины рулонов для печатей и материалов, единица измерений которых кв. метры (м2).

### Создание Ширины рулона

Чтобы создать Ширину рулона, в Справочнике в Свойствах, щелкните мышкой на Ширина рулона

![](<../../.gitbook/assets/image (2098).png>)



и нажмите кнопку "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (2521).png>)



В открывшемся окне заполните _название_ типа Ширины рулона

![](<../../.gitbook/assets/image (2365).png>)

После сохранения, тип рулона отобразится в общем списке

![](<../../.gitbook/assets/image (2358).png>)



Чтобы изменить название типа рулона, нажмите на кнопку "Редактировать"

![](<../../.gitbook/assets/image (2134).png>)



{% hint style="info" %}
Удаление типа ширины рулонов из общего списка не предусмотрено. Вы можете отредактировать его название и внести другую ширину рулона.
{% endhint %}





### Создание Ширины рулона

После создания типа Ширины рулона необходимо внести в него точную ширину листа. Для этого:

1\)  щелкните мышкой на _название_ типа ширины,

&#x20;

![](<../../.gitbook/assets/image (2379).png>)



2\) нажмите кнопку "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (481).png>)



3\) заполните параметры: _Ширина листа\*_

![](<../../.gitbook/assets/image (2947).png>)

После сохранения параметров, внесенная ширина рулона появится в списке

![](<../../.gitbook/assets/image (1203).png>)



{% hint style="info" %}
В Ширину рулона может входить неограниченное количество ширины листов.&#x20;
{% endhint %}

Например:

![](<../../.gitbook/assets/image (2602).png>)



###

### Редактирование и удаление Ширины листа

Чтобы редактировать Ширину листа, нажмите на нужную _ширину_ и в открывшейся форме внесите необходимые корректировки

![](<../../.gitbook/assets/image (2575).png>)

Чтобы удалить  ширину листа из Ширины рулона, нажмите на "Удалить" напротив нужной ширины

![](<../../.gitbook/assets/image (2115).png>)

В случае, если выбранная ширина листа используются в каком-либо материале, система выдаст соответствующее предупреждение.

![](<../../.gitbook/assets/image (2468).png>)

Ширины листа  удалится только после удаления его из материалов.
