# Формат

В Форматах вы можете установить печатные форматы и входящие в них размеры листа для материала Бумага и Ламинация.

###

### Создание Формата

Чтобы создать Формат, в Справочнике в Свойствах, щелкните мышкой на Формат

![](<../../.gitbook/assets/image (3034).png>)



и нажмите кнопку "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (1917).png>)



В открывшемся окне заполните _название_ Формата&#x20;

![](<../../.gitbook/assets/image (1661).png>)



После сохранения, формат отобразится в общем списке

![](<../../.gitbook/assets/image (2362).png>)



Чтобы изменить название формата, нажмите на кнопку "Редактировать"

![](<../../.gitbook/assets/image (1224).png>)



{% hint style="info" %}
Удаление форматов из общего списка не предусмотрено. Вы можете отредактировать его название и внести другие размеры листов.
{% endhint %}

###

### Создание Размера

После создания Формата, необходимо внести в него точные размеры листа. Для этого

1\)  щелкните мышкой на _название_ Формата,

![](<../../.gitbook/assets/image (2816).png>)

&#x20;

2\) нажмите кнопку "Добавить" в правом верхнем углу

![](<../../.gitbook/assets/image (2231).png>)



3\) заполните параметры: _Ширина_ и _Высота листа_

![](<../../.gitbook/assets/image (1652).png>)



После сохранения параметров, внесенный размер появится в списке

![](<../../.gitbook/assets/image (1680).png>)



{% hint style="info" %}
В Формат может входить неограниченное количество размеров листов.&#x20;
{% endhint %}



Например:

![](<../../.gitbook/assets/image (1184).png>)





### Редактирование и удаление Размера

Чтобы редактировать Размеры листа, нажмите на нужный _размер_ листа и в открывшейся форме внесите необходимые корректировки

![](<../../.gitbook/assets/image (3131).png>)



Чтобы удалить размеры листа из формата, нажмите на кнопку "Удалить" напротив нужного размера

![](<../../.gitbook/assets/image (2367).png>)

В случае, если выбранные размеры листов используются в каком-либо материале, система выдаст соответствующее предупреждение.

![](<../../.gitbook/assets/image (2374).png>)

Размер удалится только после удаления его из материалов.
