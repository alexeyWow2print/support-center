# Свойства

В процессе заполнения операция, доп. операций, комплектующих и товаров мы неоднократно обращались к одному из самых важных вспомогательных разделов - Свойства. Рассмотрим его подробнее.

В данном разделе заполняются:

* Форматы и размеры материала Бумага и Ламинация
* Ширина рулона (для продукции, ед. изм. которой - кв.м.)
* Плотность
* Толщина
* Доп. услуги
* Цвет
* Теги
* Свойства комплектующих
* Группировка
