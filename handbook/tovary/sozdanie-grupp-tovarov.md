# Создание групп Товаров

Системой предусмотрена возможность группировать Товары по следующему алгоритму (на примере товара Кружка).

###

### **Создание Групп товаров**

В  разделе для удобства предусмотрена группировка по товарам. Создать группу можно через соответствующую кнопку "Добавить группу".&#x20;

![](<../../.gitbook/assets/image (2088).png>)

Заполните поле _Название_ и сохраните папку.

![](<../../.gitbook/assets/image (1914).png>)

После сохранения папка появился в общем списке.

![](<../../.gitbook/assets/image (3708).png>)



### **Группировка Товаров**

Чтобы сгруппировать внесенные данные, в соответствующем товаре в поле _Группа_ выберите  нужную, и нажмите "Сохранить"

![](<../../.gitbook/assets/image (603).png>)



Товары появятся в папке.

![](<../../.gitbook/assets/image (3707).png>)

###

### **Редактирование и удаление папок**

Чтобы редактировать _название_ Группы (папки) товаров или удалить, нажмите соответствующие кнопки

![](<../../.gitbook/assets/image (2243).png>)



{% hint style="info" %}
**В случае удаления Группы, входящие в нее Товары не удаляются!** Товары попадают в общий список товаров.
{% endhint %}
