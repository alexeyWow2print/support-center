# Создание Групп/папок материалов

### **Создание Групп/папок  материалов**

В разделе для удобства предусмотрена группировка по внесенным типам бумаги.&#x20;

Создать группу можно через соответствующую кнопку «Добавить группу».&#x20;

![](<../../../.gitbook/assets/image (715).png>)



Заполните поле _Название_ и сохраните папку.

![](<../../../.gitbook/assets/image (3246).png>)



### **Группировка материала**

Чтобы сгруппировать внесенные данные, в соответствующем материале в  поле _Группа_ выберите  нужную, и нажмите «Сохранить»

![](<../../../.gitbook/assets/image (1610).png>)



### **Редактирование и удаление папок**

Чтобы редактировать _название_  Группы/папки  или удалить, нажмите соответствующие кнопки

![](<../../../.gitbook/assets/image (1050).png>)



{% hint style="info" %}
**В случае удаления Группы, входящие в нее материалы не удаляются!** Материалы попадают в общий список материалов.
{% endhint %}
