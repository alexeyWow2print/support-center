# Вкладка Свойства

**Добавление свойства материала**

![](<../../../.gitbook/assets/image (3653).png>)

В открывшемся окне заполните следующие параметры из списка:

![](<../../../.gitbook/assets/image (3154).png>)

* **РАЗМЕР** (отображаются все форматы/размеры заполненные в Справочнике)

**Для единицы измерения  ЛИСТ** в поле _Размер_ отображается размер _ширина_ х _высота_ листа.

Размеры вносятся в Справочник → раздел Свойства → Формат → кнопка "Добавить" → Сначала вносится **Формат**&#x20;

![](<../../../.gitbook/assets/image (3258).png>)

Затем, щелкните  в списке на _название_ созданного Формата&#x20;

![](<../../../.gitbook/assets/image (726).png>)

&#x20;и через кнопку "Добавит&#x44C;**"**

![](<../../../.gitbook/assets/image (1166).png>)

внесите нужные _**ширину**_**&#x20;и&#x20;**_**высоту листа**_ в ед. измерения миллиметры (мм.)

![](<../../../.gitbook/assets/image (3553).png>)

&#x20;По такому алгоритму в один Формат можно внести любое количество Размеров, не нужно создавать на каждый размер новый формат.



**Для единицы измерения КВ. МЕТР (М2) в поле размер отображается ширина рулона.**

Ширина рулона вносятся в Справочник → раздел Свойства → Ширина рулона→ кнопка "Добавить"

По аналогии с Форматом, вносится сначала **Тип ширины рулона**

![](<../../../.gitbook/assets/image (3642).png>)

А затем через щелкнув в общем списке на название, зайдите в созданный тип ширины рулона

![](<../../../.gitbook/assets/image (2648).png>)

И через кнопку "Добавить"

![](<../../../.gitbook/assets/image (3527).png>)

введите нужную _ширину_ листа/рулона. Ширина вводится в ед. измерения миллиметры (мм.)

![](<../../../.gitbook/assets/image (3526).png>)



* **ПЛОТНОСТЬ**

Плотность вносится в Справочник → Свойства → Плотность → кнопка "Добавить" →заполните поле _Значение_ и _ед. измерения_.&#x20;

После сохранения данных, внесенное значение появится в списке плотностей в Свойствах. Обращаем внимание, что плотность является обязательным параметром для заполнения, т.к. если плотность не будет указана - вес материала будет 0.

![](<../../../.gitbook/assets/image (3271).png>)

* **ТОЛЩИНА**

&#x20;Толщина вносится в Справочник → Толщина → кнопка "Добавить"→ заполните поле _Значение_ и выберите _ед. измерения_ (мм. или мкм.) &#x20;

После сохранения данных, внесенное значение появится в списке плотностей в Свойствах.&#x20;

![](<../../../.gitbook/assets/image (3416).png>)



* **ВИД ПЕЧАТИ** (Универсальная, Цифровая, Офсетная).  В случае выбора конкретного типа печати (Цифровая/Офсетная), бумага не будет доступна для другого типа печати.&#x20;



* **ТЭГ**

Тег заполняется в Справочник → Свойства → Теги → кнопка "Добавить" → заполните поле _Название_ и выберите _Тип._

![](<../../../.gitbook/assets/image (3317).png>)



**Изменение свойства материала**

Чтобы изменить Свойства материла, в правом углу напротив каждого формата нажмите кнопку "Редактировать"

![](<../../../.gitbook/assets/image (3803).png>)

\
По строкам: _Плотность, Толщина, Тэг, Вид печати_,  откроются поля для замены выбранных ранее вариантов. После внесения изменений нажмите галочку, в конце строки, чтобы подтвердить их окончательно.

![](<../../../.gitbook/assets/image (1543).png>)

**Удаление свойства материала**

Чтобы удалить свойства материла нажмите на кнопку "Удалить" напротив нужной строки характеристик.

![](<../../../.gitbook/assets/image (762).png>)

В случае, если данный формат с указанными характеристиками уже используется в продукции система выдаст сообщение: "Ошибка изменения размер материала" и предложит удалить материал из продукции.

![](<../../../.gitbook/assets/image (1547).png>)

Для окончательного удаления и нажмите кнопку "Заменить".
