# Смена доменного имени

## Общие положения

Изначально сайт предоставляется на поддомене site._wow2print.com,_ после внесения основных настроек, его необходимо переадресовать на ваш основной домен.\
\
Процесс переадресации выглядит следующим образом:

1. **Внести изменения на сайте регистратора доменных имен (reg.ru, nic.ru и т.д.)**\
   Необходимо перейти на сайт, на котором вы зарегистрировали доменное имя и в настройках доменного имени добавить/изменить ресурсные записи (2 записи) типа А:
   1. Для первой ресурсной записи укажите в полях:
      1. **Тип (Subdomain)** — значок «@»;
      2. **IP-адрес** — <mark style="color:red;">**84.252.137.194**</mark> 🇷🇺 (для сайтов в РФ) / <mark style="color:red;">**37.27.212.243**</mark>  🌍(для сайтов других стран)
   2.  Для второй ресурсной записи:

       1. **Тип (Subdomain)** — «www»;
       2. **IP-адрес** — <mark style="color:red;">**84.252.137.194**</mark> (для сайтов в РФ) / <mark style="color:red;">**37.27.212.243**</mark> (для сайтов других стран)

       Ниже показано добавление ресурсных записей на [примере сайта www.reg.ru](smena-domennogo-imeni.md#a-zapis-dobavit-izmenit-resursnuyu-zapis)<br>
2. **Дождаться обновления записей**\
   Ресурсные записи обновляются от 5 минут до 48 часов.\
   Как узнать обновились ли записи: при попытке открыть сайт, откроется пустая страница с надписью в левом верхнем углу — "Подключение не удалось".<br>
3. **Создать заявку в Сервис Деск**\
   После обновления записей, необходимо перейти в [Сервис Деск](https://tippo.okdesk.ru/) и создать заявку с названием "Переадресация домена", в описании заявки укажите доменное имя, на которое необходимо переадресовать сайт.<br>
4. **Завершить переадресацию**\
   Сотрудники TCS внесут новое доменное имя во внутреннюю базу данных. С этого момента сайт будет доступен только по новому адресу.<br>
5. **Сертификат безопасности**\
   Сертификат безопасности на сайт будет установлен в течение 1-2 рабочих дней.



{% hint style="danger" %}
После переадресации сайта на другой домен, ссылки для входа в админ-панель TCS также изменится
{% endhint %}





## А-запись. Добавить/изменить ресурсную запись



На примере сайта reg.ru

* Авторизуйтесь на сайте reg.ru, перейдите в раздел «Мои домены и услуги» и нажмите на название вашего доменного имени:

![](<../.gitbook/assets/image (3165).png>)



* Далее перейдите в «DNS-серверы и управление зоной»;

![](<../.gitbook/assets/image (844).png>)



* На открывшейся странице обратите внимание на раздел «Ресурсные записи». В нем уже имеются необходимые две записи типа «А». Нам осталось лишь прописать правильный ip-адрес. Для этого нажимаем на иконку карандаша и для каждой записи изменяем ip-адрес на 84.252.137.194 (для сайтов в РФ),  49.12.186.88 (для сайтов других стран);

![](<../.gitbook/assets/image (3356).png>)



* Сохраните изменения для двух записей;

![](<../.gitbook/assets/image (3589).png>)
