# Реализация

### Реализации

Все закрывающие документы по заказам и отгрузкам, у которых плательщиком является юридическое лицо, находятся в подразделе Реализация.&#x20;

В верхней стоке документы можно отсортировать по типу и произвести поиск:

* по номеру
* поставщику
* плательщику
* заказу
* отгрузке
* дате

И отфильтровать по типу:

* Акт оказания услуг
* Товарная накладная&#x20;
* Счет фактура



![](<../../.gitbook/assets/image (3178).png>)



###

### Просмотр закрывающих  документов

Просмотреть и распечатать закрывающие документы можно нажав на кнопку напротив каждого документа

![](<../../.gitbook/assets/image (2015).png>)



###

### Удаление реализаций

Чтобы удалить реализацию, нажмите кнопку "Удалить" напротив выбранного документа

![](<../../.gitbook/assets/image (3536).png>)
