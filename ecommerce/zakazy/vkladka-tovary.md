# Вкладка Товары

### Список товаров

Рядом с вкладкой Заказы находится вкладка Товары.

<div align="center"><img src="../../.gitbook/assets/image (1679).png" alt=""></div>



Из общего списка товаров вы можете перейти на карточку товара, щелкнув на номер.

<div align="center"><img src="../../.gitbook/assets/image (2120).png" alt=""></div>





### Работа со списком товаров

Вкладка содержит расширенный поиск по Товарам в зависимости от параметров:&#x20;

* Бумага/Материал,&#x20;
* Плотность,&#x20;
* Цветность,
* Тег материала,
* Тег товара,&#x20;
* Печать,
* Выбор пользователя.

<figure><img src="../../.gitbook/assets/расширенный фильтр товара.png" alt=""><figcaption></figcaption></figure>

Вы также можете использовать сортировку и фильтр в верхней части списка товаров.&#x20;

![](<../../.gitbook/assets/image (2038).png>)

### Настройки параметров отображения&#x20;

Вы можете настроить отображение товаров по разным параметрам, нажав на кнопку ("шестеренки")

![](<../../.gitbook/assets/image (1660).png>)



&#x20;и отметив  в открывшемся окне нужные столбцы для отображения (_номер/превью/продукт/кол-во/дата заказа/оплата/стоимость/менеджер/размер продукции/дизайнер/клиент/статус/бумага/пункт выдачи_)  &#x20;

![](<../../.gitbook/assets/image (508).png>)

### Группировка товаров по вкладкам

Товары дублируют статус заказов и  группируются по вкладкам идентичным вкладкам заказов.

![](<../../.gitbook/assets/image (1096).png>)



При изменении статуса заказа статус товара меняется на статус заказа.

Если в заказе несколько товаров и у них по разному меняются статусы (например, один товар  статус "Проверен", а другой статус "В печати"), статус заказа будет определятся по **наименьшему статусу товара**.

Товары из заказов, у которых истек срок оплаты перемещаются в папку Неоплаченные.

![](<../../.gitbook/assets/image (1857).png>)





### Экспорт данных о товарах

Информацию о товарах по выбранным параметрам можно скачать в exel-файл, нажав на кнопку "Скачать в файл".&#x20;

![](<../../.gitbook/assets/image (3573).png>)



Данные по товарам сохраняются в таблицу по параметрам выбранным в настройках параметров отображения (в Столбцах отображения).





### Формирование отгрузки&#x20;

При подключенных модулях Отгрузки,  в Товарах предусмотрена возможность формирования отгрузок из разных заказов одного клиента. Для этого нужно отметить позиции товара и нажать "Создать отгрузку".&#x20;

![](<../../.gitbook/assets/image (2645).png>)



Данные по созданным отгрузкам отражаются в&#x20;

> разделе Магазин → Отгрузки.



Статусы и вкладки отгрузок устанавливаются в&#x20;

> Настройки → Статусы → Статусы отгрузок





### Формирование сборного тиража &#x20;



![](<../../.gitbook/assets/image (1971).png>)



Правила сборных тиражей создаются в&#x20;

> Настройки → Другие настройки → Правила сборных тиражей



Статусы и вкладки сборных тиражей  устанавливаются в&#x20;

> Настройки → Статусы → Статусы сборных тиражей.



\
Данные по сборным тиражам отражаются в разделах&#x20;

> Магазин → Сборные тиражи



### Удаление товаров

Чтобы удалить товар из списка,  выберите его и нажмите кнопку "Удалить" в правом нижнем углу.

![](<../../.gitbook/assets/image (2107).png>)



После  удаления, Товар попадает во вкладку Помеченные на удаление.

![](<../../.gitbook/assets/image (2459).png>)
