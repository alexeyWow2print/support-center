# Проверка калькуляции

{% hint style="info" %}
#### Проверка калькуляции работает только при рецептной калькуляции. В случае, если в продукте использована калькуляция в таблице, отображаться будут только те операции, материалы и др. модули, которые не были включены в цену при заполнении таблицы (не выбраны бегунками перед созданием таблицы с комбинациями параметров).&#x20;
{% endhint %}



Чтобы проверить калькуляцию продукта после внесения всех модулей, нажмите кнопку "Проверка калькуляции" в правом верхнем углу.

<figure><img src="../../../../.gitbook/assets/image (3863).png" alt=""><figcaption></figcaption></figure>



Откроется форма со всеми внесенными параметрами, открытыми для отображения на сайте.

<figure><img src="../../../../.gitbook/assets/image (3864).png" alt=""><figcaption></figcaption></figure>

## Окно Проверка калькуляции

Чтобы посмотреть из цен каких модулей складывается цена на продукт, выберите параметры, по которым планируется проверка и нажмите на кнопку "Проверка калькуляции" в правом нижнем углу.&#x20;

Вы увидите полный расклад калькуляции: размер (с учетом вылетов под обрез), кол-во изделий на листе, кол-во листов бумаги, цена за операцию печать, резка, ламинация и т.д.

<figure><img src="../../../../.gitbook/assets/image (3865).png" alt=""><figcaption></figcaption></figure>



Для изменения калькуляции, вы можете пройти обратно в во вкладку калькуляции продукта, нажав "Изменить".&#x20;

Если продукт состоит из нескольких деталей, проверка калькуляции будет отображать все детали и модули, добавленные к ним.

## Окно указания параметров в ручную

В системе предусмотрено ручное добавление/изменение параметров: _Группа, Цена_ (отменяет цену в калькуляции), себестоимость (подтягивается автоматически если указана, либо равна цене) и _Вес_.

<figure><img src="../../../../.gitbook/assets/image (3866).png" alt=""><figcaption></figcaption></figure>



В случае, если у вас подключен модуль "Корпоративная платформа", по кнопке "Создать преднастройку" вы можете создать корпоративный шаблон для клиентов, использующих корпоративную платформу.
