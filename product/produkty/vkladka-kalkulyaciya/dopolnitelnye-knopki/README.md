# Дополнительные кнопки

В правом верхнем углу находятся кнопки: "Калькуляция в таблице", "Проверка калькуляции" и "Продукт на сайте".

<figure><img src="../../../../.gitbook/assets/image (359).png" alt=""><figcaption></figcaption></figure>



{% tabs %}
{% tab title="Настройки" %}
<figure><img src="../../../../.gitbook/assets/image (360).png" alt=""><figcaption></figcaption></figure>

Кнопка позволяет внести дополнительные настройки в калькуляцию продукта. Можно изменить название кнопок и добавить дополнительную кнопку "Заказ через менеджера". Подробнее в разделе [Настройки](./#nastroiki)
{% endtab %}

{% tab title="Калькуляция в таблице" %}
<figure><img src="../../../../.gitbook/assets/image (361).png" alt=""><figcaption></figcaption></figure>

Через данную кнопку вы можете внести цены для табличной калькуляции. Подробнее об этом в разделе [Калькуляция в таблице](https://support.wow2print.com/product/produkty/vkladka-kalkulyaciya/knopka-kalkulyaciya-v-tablice)
{% endtab %}

{% tab title="Проверка калькуляции" %}
<figure><img src="../../../../.gitbook/assets/image (362).png" alt=""><figcaption></figcaption></figure>

После внесения всех деталей, модулей вы можете проверить их правильность (и ценообразование), нажав на кнопку "Проверка калькуляции". Подробнее об этом в разделе [Проверка калькуляции](https://support.wow2print.com/product/produkty/vkladka-kalkulyaciya/knopka-proverka-kalkulyacii)
{% endtab %}

{% tab title="Продукт на сайте" %}
<figure><img src="../../../../.gitbook/assets/image (363).png" alt=""><figcaption></figcaption></figure>

Кнопка позволяет быстро перейти на страницу продукта на сайте. Подробнее об этом [Продукт на сайте](https://support.wow2print.com/product/produkty/vkladka-kalkulyaciya/knopka-produkt-na-saite)
{% endtab %}
{% endtabs %}
