# Модуль «Установка люверсов»

## Добавление операции Установка люверсов в калькуляцию продукта

{% hint style="info" %}
Добавить модуль можно только при условии, если в калькуляцию продукта  добавлена Деталь. Т.е. сначала вы добавляете Деталь, а затем прикрепляете к ней нужные модули.
{% endhint %}

###

### Добавление операции в калькуляцию

Чтобы добавить операцию Установка люверсов, нажмите на знак <img src="../../../../.gitbook/assets/Screenshot_14.png" alt="" data-size="original"> внизу страницы и выберите "Добавить модуль".

![](<../../../../.gitbook/assets/image (1828).png>)

В открывшемся окне выберите:

* &#x20;Деталь → укажите к какой детали  будет прикреплена операция Установка люверсов
* Типа модуля →выберите из списка →Доп. операция
* Операция → выберите Установка люверсов
* Комплектующие → выберите нужный типа комплектующих → [Люверс](../../../../handbook/komplektuyushie/lyuvers.md)

![](<../../../../.gitbook/assets/image (2543).png>)

Не забудьте нажать кнопку "Сохранить" ( <img src="../../../../.gitbook/assets/Screenshot_15 (4).png" alt="" data-size="line"> )



После сохранения, модуль операции Установка люверсов прикрепится к выбранной детали. Чтобы добавить саму операцию, нажмите кнопку "Добавить".

![](<../../../../.gitbook/assets/image (446).png>)

и выберите нужную операцию из [списка Справочника](../../../../handbook/dop.-operacii/ustanovka-lyuversov.md).

Операция Установка люверсов добавится вместе с выбранными комплектующими люверс.

![](<../../../../.gitbook/assets/image (735).png>)

Чтобы добавить еще операции нажмите на кнопку "Добавить" ( <img src="../../../../.gitbook/assets/Screenshot_17 (1).png" alt="" data-size="original"> )

### Вспомогательная операция Проклейка

Операция Установка люверсов предполагает вспомогательную операцию [Проклейка](../../../../handbook/dop.-operacii/prokleika.md), которую нужно посчитать в основной логике.&#x20;

Нажмите на <img src="../../../../.gitbook/assets/Screenshot_16 (1).png" alt="" data-size="original"> напротив операции и выберите из списка доп. операций нужную&#x20;

![](<../../../../.gitbook/assets/image (2003).png>)

Вспомогательная операцию добавится к основной:

![](<../../../../.gitbook/assets/image (2204).png>)



### Варианты выбора количества

В столбце _Выбор количества_ установите логику дальнейшего расчета количества:

* **Фиксированный**

В случае выбора данного варианта, кол-во доп операции будет всегда одинаковым на ед. продукта. Количество вносится в поле ниже.

![](<../../../../.gitbook/assets/image (1837).png>)



* **Вручную**&#x20;

При выборе Вручную, клиент может сам вносить нужное количество на сайте. Для отображения поля введения данных, включите "Отображать поле ввода".

![](<../../../../.gitbook/assets/image (2617).png>)



* **По размерам**

В этом случае, количество операций будет рассчитываться исходя из периметра изделия. Через индексы вы можете управлять параметрами ширины и высоты.

![](<../../../../.gitbook/assets/image (3542).png>)

Например, если вам нужно рассчитать люверсы по всему периметру, установите индексы ширины и высоты 2, если только по ширине, установите индекс ширины 2 и т.д..&#x20;

{% hint style="info" %}
При выборе варианта **По размерам** системой предусмотрено автоматическое добавление +1 (нулевой) операции
{% endhint %}



Для корректного расчета Установки люверсов при выборе количества "По размерам",  включите "Отображать поле ввода" и установите возможность выбора клиентом расстояние между люверсами.&#x20;

![](<../../../../.gitbook/assets/image (2570).png>)



По такому же алгоритму устанавливается выбор количества для вспомогательной операции

![](<../../../../.gitbook/assets/image (486).png>)





### Визуальное оформление Установки люверсов

В верхней части модуля вы можете оформить отображение операции на сайте для клиента

![](<../../../../.gitbook/assets/image (2260).png>)

* Переключателем "Показывать на сайте" скрыть или отобразить операцию на сайте
* Переключателем "Показывать поле Заголовок на сайте" скрыть или отобразить заголовок на сайте.
* В поле _Заголовок на сайте_ озаглавьте модуль на сайте
* В _Отображении_ выберите способ: Выпадающий список, Радио баттон, Иконками, Радио баттон с иконками.
* Информация в _Описании_ будет расположена сверху до отображения операции
* В строке _Галерея_ включить или выключить отображение на сайте картинок, загруженных в операция Установка люверсов → вкладку Фотогалерея
* В _Заголовке для поля ввода_ впишите заголовок и укажите минимальное и максимальное значение.&#x20;
* В помощью выбора _Отображать доп операции_ вы можете показать/скрыть вспомогательные операции на сайте.

Также в "Настройке доп. операций", рядом с операцией Установка люверсов предусмотрена возможность загрузить _иконку_ (она отобразится при выборе строки Отображения → Иконками) и в _Названии варианта параметра_ вписать то, что будет видно на сайте клиентам.

![](<../../../../.gitbook/assets/image (2179).png>)

![](<../../../../.gitbook/assets/image (883).png>)

![](<../../../../.gitbook/assets/image (2732).png>)

Выбранные люверсы появятся в списке

![](<../../../../.gitbook/assets/image (2966).png>)



### Выбор количества

В столбце выбор количества вы можете выбрать логику расчета количества:

* **Как в операции**

В этом случае, количество комплектующих люверс  будет рассчитываться исходя из логики основной операции (т.е. дублировать логику выбранную в столбце "Выбор количества" в  Установка люверсов).&#x20;

{% hint style="info" %}
При выборе варианта **Как в операции** системой предусмотрено автоматическое добавление +1 шт. комплектующего (люверс).
{% endhint %}

![](<../../../../.gitbook/assets/image (3204).png>)



* **Фиксированный**

Установите фиксированное количество люверсов на единицу изделия. Количество, внесенное  поле значения "Фиксированный"  будет постоянно, вне зависимости от расчета кол-ва в операции

![](<../../../../.gitbook/assets/image (1522).png>)



### Визуальное оформление Комплектующих Люверс

Настроить визуальное отображение Люверсов можно в верхней части модуля&#x20;

![](<../../../../.gitbook/assets/image (493).png>)

* Переключателями  "Показывать на сайте" выбрать отобразить/ скрыть информацию на сайте
* Выбрать показывать заголовок на сайте или нет, при помощи ползунка "Показывать заголовок на сайте"
* поле _Заголовок на сайте_ озаглавьте модуль на сайте
* В _Отображении_ выберите способ: Выпадающий список, Радио баттон, Иконками, Радио баттон с иконками.
* Информация в _Описании_ будет расположена сверху до отображения люверсов
* В строке _Галерея_ включить/выключить отображение на сайте картинок, загруженных в Комплектующие Люверс → вкладку Фотогалерея

Дополнительно вы можете настроить следующие опции:

![](<../../../../.gitbook/assets/image (3580).png>)

* **Отображать свойства** (вы можете показать свойства, заполненные в Люверс → вкладка Свойства, выбрав тип отображения: Выпадающий список, Радио баттон, Иконками, Радио баттон с иконками).
* **Отображать поле ввода** (выберите, если вам нужно чтобы какой-либо параметр устанавливался вручную)
* **Показывать группировки на сайте** (отображает выбранные группировки при условии заполнения в Справочник → Свойства → Группировки&#x20;
* **Отображать текстовое поле** (используйте, если вам нужно чтобы клиент мог внести текстовые данные в Люверсы)

В "Настройка комплектующих" вы можете:

* &#x20;загрузить _Иконку_. Если иконка здесь не загружена, автоматически подгружается картинка  комплектующего люверс.&#x20;
* &#x20;внести _Название варианта параметра_ для сайте, отличное от названия в Справочнике. В случае, если это поле не заполнено, название автоматически берется из Справочника.

![](<../../../../.gitbook/assets/image (2557).png>)
