# 🛍️ Продукция

## Общие положения

_Данный раздел позволяет управлять каталогом продукции:_ [_создать новую категорию_](kategorii.md#sozdanie-kategorii)_,_ [_редактировать существующую_](kategorii.md#redaktirovanie-kategorii)_,_ [_сортировать категории между собой_](kategorii.md#redaktirovanie-kategorii)_, размещать одну категорию в другой, создать новый продукт, редактировать существующий, сортировать продукты между собой и пр._

{% hint style="info" %}
Важно понимать, что каждая категория и каждый продукт — это отдельная страница сайта со своим контентом и настройками, в том числе SEO.

Категория продукции может отображаться калькуляцией (в виде продукта), но все же их можно друг от друга отличить:

* В адресной строке категория имеет вид — demo.wow2print.co&#x6D;**/vizitki**
* Продукт — demo.wow2print.co&#x6D;**/produkciya/vizitki.html**
{% endhint %}
