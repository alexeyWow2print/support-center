# Виджет «Форма обратной связи»

**Добавление формы обратной связи**

Чтобы добавить форму обратной связи нажмите на кнопку "Добавить" в правом углу.

![](<../../.gitbook/assets/image (3734).png>)



**Заполнение параметров**

Заполните в открывшейся форме данные:

* _Название_
* _Капча_  → Выключена/включена
* _Отображение формы_: Вертикальная или Горизонтальная
* _Отображение подписи:_ Да/Нет
* _Название формы на сайте_
* _Подпись кнопки_

![](<../../.gitbook/assets/image (1031).png>)



После сохранения, заполните поля запросов и выберите ползунками  "Отображать" /"Не отображать" на сайте и "Обязателен", если поле в форме обязательно для заполнения.

&#x20;В "Маске" вы можете настроить маску для телефона, запрашиваемого в обратной связи.

![](<../../.gitbook/assets/image (3418).png>)



**Код для установки на сайта**

Внизу формы скопируйте код для установки на сайт и вставьте его в разделе Контент → Страницы.&#x20;

![](<../../.gitbook/assets/image (1648).png>)

Для этого выберите нужную страницу, в редакторе выберите режим исходного кода&#x20;

![](<../../.gitbook/assets/image (2691).png>)

вставьте скопированный код  и сохраните.

Можно также вставить данный код в разделе Продукция. Выберите нужный товар, зайдите в редактор страницы продукции, выберите режим исходного кода, вставьте скопированный код и сохраните.
