# Виджет «Рекомендуемая продукция»

## Общий вид

![](<../../.gitbook/assets/image (3663).png>)



Куда следует размещать виджет «Рекомендуемая продукция»:

1. **На страницу каждого продукта** из вашего каталога\
   Расположите его в контент под калькуляцией.<br>
2. **На страницу Корзины**\
   Виджет будет отображаться как при оформлении заказа, так и в пустой корзине.



{% hint style="info" %}
Данный виджет визуально напоминает виджет «Каталог товаров», основное отличие заключается в том, что виджет «Каталог товаров» отображает Категории продуктов, а виджет «Рекомендуемая продукция» отдельные Продукты.
{% endhint %}





## Как создать?

Чтобы создать виджет «Рекомендуемая продукция», в админ-панели сайта войдите в раздел «_Контент → Виджеты»_, нажмите на кнопку «Добавить» в правом верхнем углу. В открывшемся окне найдите виджет «Рекомендуемая продукци&#x44F;_»_ и нажмите «Создать».





## Параметры





### Общие

Перед вами откроется форма с возможностью выбрать параметры виджета.

![](<../../.gitbook/assets/image (3588).png>)



Заполните поля и выберите параметры:

* **Название виджета**\
  Внутреннее название для админ-панели. Нигде не отображается.<br>
* **Тип устройства**
  * Универсальный — виджет будет отображаться на всех устройствах;
  * Для десктопа — отображение будет только на компьютере/ноутбуке;
  * Для мобильных устройств — отображение только на мобильных устройствах.<br>
*   **Заголовок**\
    Заголовок типа H2, отображается над виджетом.


* **Количество в ряд**\
  Можно выбрать 3, 4 или 6 элементов в ряд.<br>
* **Количество строк**\
  Можно выбрать 1, 2 или 3 строки.\
  Данный виджет отображает фиксированное количество строк, если элементов больше, чем может отобразить виджет, то отобразятся лишь первые из списка включенных продуктов.<br>
* **Категории и продукты**\
  Виджет НЕ отображает категории продуктов, необходимо выбрать продукты, которые будут отображаться на сайте.



{% hint style="warning" %}
Не забудьте активировать виджет после создания. Это можно сделать в разделе «Контент → Виджеты», путем переключения бегунка в состояние Вкл.
{% endhint %}

## Порядок установки (2 вар.)





### 1 вариант — Через вставку кода

После сохранения всех параметров, скопируйте «Код для установки на сайт».

![](<../../.gitbook/assets/image (3407).png>)



Перейдите на нужную страницу или продукт, в режиме исходного кода вставьте код виджета в то место, которое необходимо.\
Готово!

![](<../../.gitbook/assets/Вставка кода 1.0 (2).gif>)





### 2 вариант — Через редактор страниц

Перейдите в раздел «Контент → Наполнение сайта → Страницы» нажмите на название страницы. Вы окажитесь в редакторе страниц.\
Слева выберите необходимый виджет и вставьте в поле правее в нужном порядке.\
Готово!

![](<../../.gitbook/assets/Виджет в конструкторе страниц (1).gif>)
