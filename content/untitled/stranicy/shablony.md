# Шаблоны

Шаблоны - удобный инструмент для массового создания посадочных страниц.

Если вам необходимо создать большое количество посадочных страниц по одному продукту на множество городов, воспользуйтесь Шаблонами.

## Шаблоны

### Подготовка информации

До создания шаблона необходимо определиться:

* Какая информация на посадочной странице будет изменяемая;
* На какие города/населенные пункты будут созданы посадочные страницы.

На основе этих данных необходимо создать excel-файл, в котором: первая строка — теги, все последующие — данные городов.&#x20;

Каждая новая строка — новый город. По данным каждой строки (кроме первой) создается посадочная страница.

{% hint style="danger" %}
В первой строке excel-файла располагаются теги
{% endhint %}

Пример заполнения таблицы:

![Excel-файл с тегами](<../../../.gitbook/assets/image (3119).png>)



### Теги

Теги позволяют, на основе одного шаблона, создавать множество страниц со своим уникальным контентом.

Шаблон создается на всю текстовую информацию страницы, к ней относится:

* URL;
* Заголовок — Title;
* Ключевые слова — Keywords;
* Описание — Description;
* Контент страницы.

Во всем тексте шаблона необходимо размещать определенные теги. Теги — это переменные, на место которых будут подставляться те или иные меняющаяся данные. Тег пишется на латинице и заключается в фигурные скобки "{" "}".

Меняющиеся данные — это уникальная информация для каждого города/региона.&#x20;

###

### Создание шаблона

Чтобы создать шаблон, необходимо нажать на кнопку "Добавить" (правый верхний угол).

Следом необходимо настроить шаблон:

* _Название_ — название самого шаблона;
* _Url_ — url посадочной страницы;
* _Заголовок_ — заголовок посадочной страницы;
* _Контент_ — контент посадочной страницы;
* _Ключевые слова_ — ключевые слова посадочной страницы;
* _Описание_ — описание посадочной страницы;
* _Файл_ — файл таблицы с тегами в формате excel.

Пример оформления шаблона:

![Настройки шаблона](<../../../.gitbook/assets/image (3132).png>)

{% hint style="info" %}
В блоке "Контент" может располагаться html-код сверстанной посадочной страницы, в тексте которого также могут использоваться теги.
{% endhint %}

После внесения всей информации, необходимо загрузить таблицу с тегами в поле "Файл". Следом Сохраните изменения.&#x20;

###

### Посадочные страницы

После сохранения шаблона, рядом с кнопкой "Сохранить" появится новая кнопка "Обновить страницы", нажмите на нее, чтобы создать посадочные страницы.

![Кнопка "Обновить страницы"](<../../../.gitbook/assets/image (1791).png>)

Необходимо подождать какое-то время, так как сайт создает посадочные страницы, чем больше информации в таблице с тегами, тем больше времени это может занять.

После этого станет доступна новая вкладка "Страницы", на ней располагаются все созданные посадочные страницы.

![Вкладка "Страницы"](<../../../.gitbook/assets/image (924).png>)

На этой вкладке мы можем Вкл/Выкл конкретную посадочную страницу.

![Список всех посадочных страниц](<../../../.gitbook/assets/image (919).png>)
