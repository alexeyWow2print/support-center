# Карты

### Яндекс.Карты

Чтобы активировать интеграцию с Яндекс.Карты, включите ползунок в Настройки _→_ Интеграции _→_ Карта _→_ Яндекс.Карты и подтвердите изменение статуса интеграции, нажав на кнопку "Изменить".



![](<../../.gitbook/assets/image (768).png>)



###

### Google Maps

Чтобы получить Google Maps API Key необходимо иметь учетную запись Google.&#x20;

Далее заходим на [https://cloud.google.com/maps-platform/](https://cloud.google.com/maps-platform/) и нажимаем на кнопку на кнопку «Get Started».&#x20;

![](<../../.gitbook/assets/image (1613).png>)



В появившемся окне выберите _Страну_ и согласитесь с "Условиями пользования"

![](<../../.gitbook/assets/image (2064).png>)



Заполните страницу сведений о клиенте, включая _Способ оплаты_ (оплата будет взиматься только после активации платной подписки). После введения всех данных продолжите, нажав на "Начать бесплатный пробный период".

![](<../../.gitbook/assets/image (2811).png>)



Выберите из продуктов Google Maps Platform  пункт "Карты"

![](<../../.gitbook/assets/image (2099).png>)



Включите предложенный API, нажав на кнопку "Включить"

![](<../../.gitbook/assets/image (654).png>)



После завершения настройки, скопируйте ключи API

![](<../../.gitbook/assets/image (2127).png>)



Скопированный ключ вставьте в Настройки _→_ Интеграции _→_ Карта _→_ Google Map.

![](<../../.gitbook/assets/image (967).png>)



Активируйте интеграцию с помощью ползунка и подтвердите изменение статуса интеграции.

![](<../../.gitbook/assets/image (2090).png>)
