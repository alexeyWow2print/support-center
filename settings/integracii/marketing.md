# Маркетинг

В системе вы можете пройти интеграцию с такими маркетинговыми инструментами, как сервисы рассылок "Unisender", "SMS Aero" или "SendPulse".

![](<../../.gitbook/assets/image (1572).png>)

###

###

### Unisender&#x20;

Чтобы настроить интеграцию с данным сервисом, необходимо пройти регистрацию на сайте ([https://new.unisender.com/ru/](https://new.unisender.com/ru/))

После регистрации и настройки всех параметров, в Личном кабинете → Настройки аккаунта&#x20;

![](<../../.gitbook/assets/image (2343).png>)





во вкладке Интеграции и API,  поле _Ключ доступа API_, нажмите "Показать полностью" и введите пароль от аккаунта. Скопируйте отобразившийся ключ.

![](<../../.gitbook/assets/image (601).png>)



В Настройки → Интеграции →Маркетинг →Unisender вставьте скопированный ключ в поле _Ключ доступа_ и выберите вариант списка клиентов из  из выпадающего списка _Рассылка по умолчанию._ &#x20;

В списке отображаются все созданные в личном кабинете Unisender списки контактов.

![](<../../.gitbook/assets/image (600).png>)

Напротив поля _Рассылка по умолчанию_ через кнопку "Добавить" можно добавить списки контактов.&#x20;

Созданный в системе список отобразится в личном кабинете Unisender, где вы сможете внести адреса e-mail.

Чтобы активировать интеграцию, передвиньте переключатель вправо и подтвердите изменение статуса интеграции.

![](<../../.gitbook/assets/image (2748).png>)





### SMS Aero

Для подключения интеграции с сервисом массовых рассылок, зарегистрируйтесь на сайте [https://smsaero.ru/](https://smsaero.ru/)

После регистрации в Настройках личного кабинета SMS Aero&#x20;

![](<../../.gitbook/assets/image (1991).png>)



скопируйте API-ключ и вставьте его в Настройки →Интеграции → Маркетинг → SMS Aero → поле _Ключ доступа_

![](<../../.gitbook/assets/image (3617).png>)



{% hint style="info" %}
**Важно!** Значения поля _Подписи_ в интеграции должны совпадать с _подписью_ в Личном кабинете SMS Aero (запрашивается в Личном Кабинете SMS Aero, во вкладке Подписи)
{% endhint %}

Укажите канал рассылки (в соответствии с вашими настройками и тарифами SMS Aero [https://smsaero.ru/blog/post/kak-vibrat-kanal-rassylki/](https://smsaero.ru/blog/post/kak-vibrat-kanal-rassylki/))

Настройте _Шаблоны отправки уведомлений_ о новом заказе, о изменении заказа, об оплате заказа.

<br>

### SendPulse

Для подключения **SendPulse** зарегистрируйтесь на сайте сервиса [https://sendpulse.com/ru](https://sendpulse.com/ru).

После регистрации, зайдите в Настройки аккаунта → вкладка API →  передвиньте ползунок Активировать REST API →  сохранить.&#x20;

![](<../../.gitbook/assets/image (813).png>)



После сохранения будут доступны ID и Secret.&#x20;

В Интеграции→ Маркетинг→ SendPulse → передвиньте ползунок и активируйте интеграцию.

![](<../../.gitbook/assets/image (3386).png>)



После активации, щелкните на названии и внесите Secret и ID из личного кабинета  SendPulse открывшиеся поля:

![](<../../.gitbook/assets/image (3546).png>)



и **ОБЯЗАТЕЛЬНО** выберите группу в поле "Рассылка по умолчанию".&#x20;

В поле отобразится список всех рассылок из кабинета SendPulse. Вы также может добавить новую группу через кнопку "Добавить". Созданная группа автоматически отобразится и в личном кабинете SendPulse.

![](<../../.gitbook/assets/image (3395).png>)
