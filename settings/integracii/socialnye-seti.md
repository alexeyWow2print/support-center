# Социальные сети

Интеграцию с социальной сетью Instagram можно настроить в &#x20;

> Настройки → Интеграции → Социальные сети → Instagram

![](<../../.gitbook/assets/image (2031).png>)



**Инструкция по интеграции Instagram:**

**1.** Необходимо создать аккаунт разработчика Facebook. [https://developers.facebook.com/](https://developers.facebook.com/)          \
&#x20;   1.1. Необходимо иметь аккаунт на facebook.com/ и использовать его для входа на [https://developers.facebook.com/](https://slack-redir.net/link?url=https%3A%2F%2Fdevelopers.facebook.com%2F). Если у вас нет аккаунта — создайте его

**2.** Следовать инструкциям [https://developers.facebook.com/docs/instagram-basic-display-api/getting-started](https://slack-redir.net/link?url=https%3A%2F%2Fdevelopers.facebook.com%2Fdocs%2Finstagram-basic-display-api%2Fgetting-started):   &#x20;

&#x20;   2.1. Создать приложение Facebook\
&#x20;   2.2. Настройки → Основное → прокрутите до конца страницы и в самом низу нажмите Добавить платформу → Веб-сайт →Указать сайт → Сохранить изменения\
&#x20;   2.3. Нажмите Продукты(Товары) на знак «+», найдите карточку Instagram  и нажмите на кнопку Настроить\
&#x20;   2.4. Выберите Базовое представление(Basic Display) (слева в меню Instagram), прокрутите до конца страницы и нажмите Создать приложение(Create New App)\
&#x20;   2.5. Укажите название приложения Facebook, которое вы только что создали (ID приложения введённое ранее)\
&#x20;   2.6. Теперь на этой странице прокрутите немного ниже и пропишите настройки приложения:

&#x20;а) **Клиентские настройки OAuth**

&#x20; Действительные URI переадресации для Oauth — адрес вашего сайта + /api/index/instagram (Пример: [https://demo.wow2print.com/](https://slack-redir.net/link?url=https%3A%2F%2Fdemo.wow2print.com%2F)api/index/instagram)     &#x20;

&#x20; б) **Отменить авторизацию**

&#x20; Деавторизация URL обратного вызова — адрес вашего сайта ([https://demo.wow2print.com/](https://slack-redir.net/link?url=https%3A%2F%2Fdemo.wow2print.com%2F))

&#x20; в) **Запросы на удаление данных**

&#x20; URL запроса на удаление данных— адрес вашего сайта ([https://demo.wow2print.com/](https://slack-redir.net/link?url=https%3A%2F%2Fdemo.wow2print.com%2F))

&#x20;  г) Сохранить (в правом нижнем углу окна браузера – Сохранить изменения)

\
&#x20;**3.** Добавьте тестового пользователя Instagram:

&#x20;3.1. Выберите Роли (слева) → Роли и прокрутите до раздела "**Тестировщики Instagram**". Нажмите Добавить тестировщиков Instagram (Instagram Testers), введите ваше имя пользователя Instagram и отправьте приглашение\
3.2. Откройте новое окно браузера, перейдите на сайт [www.instagram.com](https://slack-redir.net/link?url=http%3A%2F%2Fwww.instagram.com) и войдите в приглашенный аккаунт Instagram. Выберите Значок профиля → Редактировать профиль →Приложения и сайты (слева 3 пункт) → Приглашения тестировщиков и примите приглашение

&#x20;**4.** Вернитесь на вкладку developers.facebook → (Продукты(Товары) → Instagram → Базовое представление(Basic Display). Скопируйте ID приложения Instagram и вставьте в интеграции Instagram (в админ-панели вашего сайта) в "Идентификатор приложения",  Секрет приложения Instagram (нажмите «Показать») в "Ключ приложения" и введите аккаунт вашей типографии в instagram

&#x20;**5.** Создайте виджет instagram ( Контент → Виджеты → Добавить → Соц. Сети. Последние публикации → пропишите настройки → "Сохранить"), скопируйте код и вставьте на нужную страницу. **Не забудьте включить виджет!**

&#x20;**6.** Откройте страницу на сайте, где есть виджет. Откроется окно авторизации instagram → авторизуйтесь и дайте доступ к медиа приложению.
