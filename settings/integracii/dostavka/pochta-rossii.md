# Почта России

### &#x20;1. Регистрация личного кабинета Почты России и получение доступов для интеграции с TCS

Для получения доступов для интеграции Почты России с TCS необходимо:

* зарегистрироваться на сайте [https://otpravka.pochta.ru/](https://otpravka.pochta.ru/)&#x20;
* заключить договор оказание услуг с Почтой России или принять оферту
* получить доступ в личный кабинет Почты России

Для подключения услуги «Отслеживания отправлений», после создания личного кабинета, необходимо перейти на сайт [https://tracking.pochta.ru](https://tracking.pochta.ru/) нажать на «Получить доступ» (Рис.1).

После подключения в разделе «Настройки доступа» будет указан логин, пароль будет выслан на почту.

### 2. Настройка интеграции в TCS

Для добавления интеграции на сайт необходимо перейти в админ-панели в раздел Настройки → Интеграции → Доставка → выбрать вариант доставки «Почта России»

<br>

В интеграции заполняются следующие данные (Рис.2):

* в поле «Название» указывается произвольное название интеграции для вашего удобства
* в поле «Логин от личного кабинета» указывается логин вашего личного кабинета на https://otpravka.pochta.ru<br>
* в поле «Пароль от личного кабинета» указывается пароль вашего личного кабинета на https://otpravka.pochta.ru<br>
* в поле «Токен авторизации» указывается  токен авторизации из личного кабинета Почты России. Информация по токену находиться в личном кабинете Почты России (https://otpravka.pochta.ru/) в разделе Настройки → API → токен авторизации &#x20;
* в поле «Инн-договор» указывается данные в формате номер ИНН-номер договора (инн-договор используется, если между Почтой России и типографией заключен договор для предоставления индивидуальных цен на услуги доставки)
* в поле «Индекс пункта отправления» указывается индекс, который отображается в личном кабинете Почты России в разделе «Настройки сервиса»
* в поле «Логин трекинга» берется в личном кабинете
* в поле «Пароль для трекинга» указывается пароль, который был выслан на почту при оформлении услуги «Отслеживания отправлений»

<figure><img src="../../../.gitbook/assets/Почта россии 2.png" alt=""><figcaption><p>Рис.2. Поля для заполнения интеграции.</p></figcaption></figure>

После того, как интеграция настроена в корзине для клиента появится выбор доставки Почта России (в соответствии с подключенным тарифом).

### 3. Создание посылки

Для передачи заказа в личный кабинет Почты России нужно:

* перейти заказ, оформленный на доставку Почты России (в системе TCS)
* создать посылку (нажать на кнопку «Доставка» → «Создать/Обновить посылку»)
* перейти в личный кабинет Почты России и найти нужный заказ (Рис.3).

<figure><img src="../../../.gitbook/assets/Почта России 3.png" alt=""><figcaption><p>Рис.3.Отображение заказа в личном кабинете Почты России</p></figcaption></figure>

**3.1. Бланк адресного ярлыка на посылку/бандероль**

На заказ в личном кабинете Почты России формируется бланк адресного ярлыка на посылку/бандероль (Рис.4) ф.7-п.

Для того,чтобы открыть данный бланк нужно:

* в личном кабинете Почты России нажать на три точки рядом с заказом - откроется подменю
* выбрать вариант «Распечатать бланки» (Рис.5)

<figure><img src="../../../.gitbook/assets/Почта России 4.png" alt=""><figcaption><p>Рис.5. Выбор варианта «Распечатать бланк»</p></figcaption></figure>

* в новом окне откроется бланк
