## Для того, чтобы настроить данную интеграцию, вам необходимо:



**1)** заключить договор как _интернет-магазин_ с транспортной компанией "СДЭК" (во время заполнения анкеты на сайте [https://cdek.ru/](https://cdek.ru/), проставьте галочку на данный пункт).



**2)** после заключения договора, запросите ключи для интеграции, которые вам вышлют в обратном письме (подробнее [https://cdek.ru/integration](https://cdek.ru/integration) → вкладка "Часто задаваемые вопросы" →  вопрос "Где я могу найти тестовые и боевые ключи?")



**3)** полученные ключи необходимо внести в Настройки → Интеграции → Доставка → CDEK.&#x20;

В случае, если требуется добавить интеграцию, нажмите на <img src="../../../.gitbook/assets/Screenshot_1 (30).png" alt="" data-size="line"> → Добавить новую интеграцию  → выберите из списка CDEK → кнопка "Подключить"

![](<../../../.gitbook/assets/image (2411).png>)



* Заполните поле _Название\*_ и введите полученные ключи:

![](<../../../.gitbook/assets/image (3268).png>)

* Выберите свой город отправки и укажите в поле _Код ПВЗ для привоза_ код вашего пункта приема. Посмотреть код можно на сайте CDEK, в разделе Пункты CDEK [https://www.cdek.ru/ru/offices](https://www.cdek.ru/ru/offices)

<figure><img src="../../../.gitbook/assets/Screenshot_1 сдек.png" alt=""><figcaption></figcaption></figure>

* По желанию можете проставить _наценку_ или _скидку_ на стоимость доставки

<figure><img src="../../../.gitbook/assets/Screenshot_сдек 2.png" alt=""><figcaption></figcaption></figure>

* Включите тарифы, которые будут доступны для клиентов на сайте (там же, в настройках интеграции):

как правило, тарифы для интернет-магазинов — "Посылка склад-склад" и "Посылка склад-дверь", остальное по желанию (или по договору с транспортной компанией).

![](<../../../.gitbook/assets/image (993).png>)

\
При включении тарифов появится окно с настройками допустимого веса.&#x20;

![](<../../../.gitbook/assets/image (3390).png>)

\
Вес "от" желательно выставить "0", чтобы клиент мог оформить доставку продукта весом даже 1 грамм.\
Вес "до" может быть ограничен самой транспортной компанией (в зависимости от тарифа), значения 30-50 кг будут оптимальными.

После включения тарифов доставки, сохраните настройки самой интеграции.

\
**4)** активируйте выбранные в интеграции способы доставки на сайте. Для этого в Настройки → Доставка → Доставка на сайте → отсортируйте по Типу "CDEK" и включите те способы доставки, которые выбрали ранее в настройках интеграции.

![](<../../../.gitbook/assets/image (3400).png>)
