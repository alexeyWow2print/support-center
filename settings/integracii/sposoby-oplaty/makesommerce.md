# Makeсommerce

### 1. Регистрация в личном кабинете Makecommerce и получение ключей доступа

\
Для получения ключей доступа к интеграции необходимо пройти регистрацию на странице [https://merchant.maksekeskus.ee/sign-up.html?locale=en](https://merchant.maksekeskus.ee/sign-up.html?locale=en).

В личном кабинете Makeсommerce будут сформированы «Shop ID» и «Secret Key».



### 2. Настройка интеграции в TCS



Для настройки интеграции необходимо перейти в раздел Настройки → Интеграции → Способы оплаты → [makecommerce](https://test.tippo.ru/backend/payment/payment/update/22)

В интеграции заполняются следующие данные:

* в поле «Название» указывается название интеграции, которое будет отображаться на сайте при выборе оплаты в «Корзине» (Рис.1).
* в поле «Описание» можно вписать любое описание способа оплаты, оно также отображается на сайте в  «Корзине» при выборе оплаты (Рис.1).

<figure><img src="../../../.gitbook/assets/маккомерц 1.png" alt=""><figcaption><p>Рис.1. Отображение «Названия» и «Описания» интеграции.</p></figcaption></figure>

* в разделе «Компания-продавец» из списка выберите нужную компанию
* выберите для каких клиентов будет доступна оплата заказов через интеграцию («Включен в админ-панели», «Способ оплаты для юридических лиц»,«Способ оплаты для физических лиц»).
* значение «Shop ID» нужно вставить в поле «merchantId»
* значение «Secret Key» нужно вставить в поле «Ключ кабинета(private key)»
* выберите режим оплаты - Тестовый режим (выбирается для проверки работоспособности интеграции) / Рабочий режим
