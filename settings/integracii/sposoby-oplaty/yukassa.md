# ЮKassa

Перед подключением платежной системы, рекомендуем ознакомиться с требованиями к сайту  [https://yookassa.ru/docs/support/payments/onboarding/arrangement ](https://yookassa.ru/docs/support/payments/onboarding/arrangement)

### Шаг 1. Регистрация на сайте поставщика услуг

Зарегистрируйтесь на сайте [https://yookassa.ru/](https://yookassa.ru/)

Заполните необходимые данные: реквизиты для получения денежных средств, данные о руководителе, бенефициарах и т.д. и отправьте на проверку специалистам ЮKassa.

Подробнее можно почитать в документации плтженой системы [https://yookassa.ru/docs/support/merchant/payments/implement/start](https://yookassa.ru/docs/support/merchant/payments/implement/start) или уточнить у технической поддержки ЮKassa.

После проверки соответствия сайта требованиям и данных юридического лица, в Личном кабинете ЮKassa появится форма договора/заявления на подписание.

Настроить интеграцию можно будет после заключения договора.

### &#x20;Шаг 2.Настройка магазина

После того как договор будет подписан и магазин активирован, в личном кабинете ЮKassa появятся все необходимые данные для интеграции с сайтом ТКС.

{% hint style="info" %}
**Внимание!** Если у вас подключен магазин на старом протоколе HTTP, то нужно обратиться в техническую поддержку ЮKassa, и попросить сменить на новый протокол API. \
По умолчанию при регистрации подключается новый протокол API, поэтому вы также можете создать новый магазин в ЮKassa без обращения в тех. поддержку.
{% endhint %}

**Идентификатор магазина** ЮKassa находится во вкладке Настройки → Магазин

<figure><img src="../../../.gitbook/assets/Screenshot_1 (42).png" alt=""><figcaption></figcaption></figure>

**Секретный ключ магазина** можно найти во вкладке «Интеграция» в разделе «Ключи API». Для его получения сначала необходимо создать новый ключ. Для этого после входа в этот раздел вам будет доступна кнопка «Выпустить ключ».

<figure><img src="../../../.gitbook/assets/Ключ АПИ Юкассы.png" alt=""><figcaption></figcaption></figure>

Для подтверждения выпуска ключа вы получите код на указанный номер телефона. После ввода этого кода у вас будет сгенерирован ключ.

{% hint style="info" %}
**Обязательно** скопируйте его и сразу вставьте в интеграции на вашем сайте. Учтите, что позже скопировать ключ не удастся в целях безопасности.
{% endhint %}

<figure><img src="../../../.gitbook/assets/подтверждение выпуска ключа.png" alt=""><figcaption></figcaption></figure>

### Настройка уведомлений об оплате

Чтобы в заказе на сайте отобразился статус «Оплачен», выполните следующие шаги: В интеграции с ЮKassa скопируйте адрес из поля «Ссылка для HTTP уведомлений платежной системы» на вашем сайте.

<figure><img src="../../../.gitbook/assets/ссылка для HTTP уведомлений.png" alt=""><figcaption></figcaption></figure>

Затем, скопированную ссылку вставьте в личном кабинете ЮKassa в Интеграция → HTTP-уведомления&#x20;

<figure><img src="../../../.gitbook/assets/HTTP уведомления.png" alt=""><figcaption></figcaption></figure>

Нажмите кнопку "Изменить настройки" → в поле URL для уведомлений вставьте ссылку → поставьте все галочки в «Выбранные события» → нажмите Сохранить

<figure><img src="../../../.gitbook/assets/настройка уведомлений.png" alt=""><figcaption></figcaption></figure>

Так же не забудьте в интеграции на сайте ТКС указать свою систему налогообложения

<figure><img src="../../../.gitbook/assets/выбор НДС.png" alt=""><figcaption></figcaption></figure>

И ставку НДС по которой работаете, что бы в чеке была передана корректная информация.

<figure><img src="../../../.gitbook/assets/image (3869).png" alt=""><figcaption></figcaption></figure>

#### Признак способа расчёта

Признак полного расчёта в чеке нужен для указания на полную оплату товара или услуги (с учётом аванса или предоплаты) в момент получения клиентом товара или услуги.

В процессе интеграции с ЮKassa в каждом чеке по умолчанию передаётся **признак полного расчёта**.

В ЮKassa частичная предоплата, аванс и кредит не поддерживаются
