# Сувенирная продукция

В системе предусмотрены интеграции со следующими поставщиками сувенирной продукции: Проект 111 ([https://gifts.ru](https://gifts.ru/)) и Нappy gifts ([https://happygifts.ru](https://happygifts.ru/))

![](<../../.gitbook/assets/image (3108).png>)





### Проект 111

Для того, чтобы подключить на сайт интеграцию необходимо обратиться к поставщику [https://gifts.ru](https://gifts.ru/) для заключения договора и регистрации Личного кабинета на сайте поставщика.    &#x20;

После регистрации Личного кабинета на сайте поставщика, синхронизируйте Проект 111 с сайтом:\
1\.      Необходимо зайти в ЛК на сайте Проекта 111, во вкладку XML Выгрузка\
2\.      Добавить ip адрес - 78.46.64.162\
3\.      Прописать адрес вашего сайта - https://demo.wow2print.com/ \
4\.      Из этой же вкладки нам необходимы данные Логин (имеет вид \*\*\*\*\*\_xmlexport) и пароль\
5\.      Возвращаемся в админ-панель сайта в раздел Интеграции → Сувенирная продукция → Проект 111\
6\.      Ввести Логин и Пароль из пункта 4\
7\.      Сохранить\
8\.      Появится новая кнопка – "Синхронизировать", нажать на нее. Каталог выгружается около 1-2 часов.\
9\.      По прошествии этого времени необходимо зайти в Админ-панель сайта → Контент → Виджеты\
10\.  Создать [виджет сувенирной продукции](/broken/pages/-M8JtVTXE8hHsNNHSO0f)                \
11\.  Прописать название виджета\
13\.  Страница виджета – url страницы, на которой будет располагаться виджет\
14\.  Название для хлебных крошек – вписать название, которое будет отображаться в хлебных крошках\
15\.  Проставить галочку на нужном проекте.\
16\.  Сохранить\
17\.  Появится Код виджета, который необходимо вставить в код нужной страницы.



**Администрирование интеграции Проект 111**

Чтобы включить интеграцию, передвиньте переключатель вправо и подтвердите изменение статуса интеграции.

![](<../../.gitbook/assets/image (2953).png>)



Чтобы отключить интеграцию, передвиньте переключатель влево и снова подтвердите изменение статуса интеграции, нажав кнопку "Изменить"

![](<../../.gitbook/assets/image (1133).png>)

{% hint style="info" %}
Важно! Интеграция с Проект 111 дает возможность отобразить продукцию поставщика на сайте при помощи виджета. Автоматическая передача данных о заказах с сайта в личный кабинет Проект 111 отсутствует.&#x20;
{% endhint %}





### Happy Gift

Чтобы подключить интеграцию, в Настройк&#x438;_→_ Интеграции _→_&#x421;увенирная продукция, передвиньте переключатель в интеграции и подтвердите изменение его статуса.

![](<../../.gitbook/assets/image (2647).png>)



Щелкните мышкой на названии интеграции и в появившейся форме заполните поле _Кол-во дней доставки_  и нажмите кнопку "Сохранить"

![](<../../.gitbook/assets/image (632).png>)



Чтобы отключить интеграцию, передвиньте переключатель влево и подтвердите изменение статуса интеграции через кнопку "Изменить"

![](<../../.gitbook/assets/image (1466).png>)

{% hint style="info" %}
Важно!  Интеграция с Нappy gifts дает возможность отобразить продукцию поставщика на сайте при помощи виджета. Автоматическая передача данных о заказах с сайта в личный кабинет Нappy gifts отсутствует.
{% endhint %}

{% hint style="info" %}
В связи с большим количеством отображаемой продукции поставщика, интеграция может занять некоторое время.
{% endhint %}
