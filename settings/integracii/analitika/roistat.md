# Roistat

## **Настройка**

В Настройки → Интеграции → Аналитика вы можете подключить интеграцию с системой сквозной бизнес-аналитики **Roistat**&#x20;

![](<../../../.gitbook/assets/image (2146).png>)



Для этого выполните следующие действия:

1.1. В вашем личном кабинете Roistat перейдите в "**Мой Профиль"** (Иконка справа)\
1.2. В меню  слева выберете "**API**". На странице появится "**Секретный API ключ**". Нажмите "**Показать**" и скопируйте ключ.

<figure><img src="../../../.gitbook/assets/image (301).png" alt=""><figcaption></figcaption></figure>

\
1.3. Вставьте секретный ключ  в интеграции ТКС в поле "**Ключ доступа**"\
1.4. Нажмите "**Сохранить**"\
1.5. После сохранения в интеграции появится выбор **проекта** в Roistat. Выберите проект, в который будете сохранять данные. Нажмите "Сохранить"\
1.6. Далее, после сохранения, в интеграции появится поле "**id статуса "Оплачен**"". Впишите в него любой id для статуса оплачен, например "**paid**". Если id будет пустым — статус оплаты не будет передаваться в Roistat\
1.7. Нажмите "**Сохранить**"

<figure><img src="../../../.gitbook/assets/image (302).png" alt=""><figcaption></figcaption></figure>

\
2.1 В Roistat перейдите в проект, который был выбран в интеграции.\
2.2. В меню слева перейдите в Настройки\
2.3. На странице настроек перейдите в "<mark style="background-color:yellow;">**Основные**</mark><mark style="background-color:yellow;">" → "</mark><mark style="background-color:yellow;">**Код счетчика**</mark>". Скопируйте код "**\<script> ......**"\
2.4. В админ-панели перейдите в <mark style="background-color:yellow;">Настройки → Сайт → Вставка кода в \<HEAD> И \<BODY> сайта</mark>. Вставьте скопированный код в поле **"body сайта"** и нажмите "Сохранить".\
2.5. В Roistat на странице кода счётчика проверьте правильность установки кода. Введите адрес вашего сайта в поле **Для проверки правильности установки счётчика** и нажмите "**Проверить счётчик**"&#x20;

<figure><img src="../../../.gitbook/assets/image (303).png" alt=""><figcaption></figcaption></figure>

3.1. Далее на странице настроек проекта Roistat перейдите в "**Основные**" → "**Настройки интеграций**"\
3.2. Нажмите "**Добавить интеграцию**", выберите **API**. Интеграция API появится в поле CRM (снизу) → нажмите "**Настроить интеграцию**" (шестерёнка)

<figure><img src="../../../.gitbook/assets/image (305).png" alt=""><figcaption></figcaption></figure>

\
3.3. В интеграции скопируйте **Ссылку на сделку** и **Ссылку на контакт** и в ставьте в настройках API в Roistat в поля **Ссылка на сделку в CRM** и **Ссылка на контакт в CRM** соответственно и сохраните.

<figure><img src="../../../.gitbook/assets/image (308).png" alt=""><figcaption></figcaption></figure>



## **Основные возможности Roistat**

### **Анализ данных и визуализация**

Roistat предлагает инструментарий для анализа эффективности рекламных кампаний, объединяя все ключевые данные в одной платформе. Это позволяет бизнесу получить полную картину взаимодействия клиентов с сайтом и оценить вклад каждого рекламного канала.

В разделе <mark style="background-color:yellow;">**Аналитика → Отчёты**</mark> вы найдёте детализированную информацию о поведении посетителей. Система автоматически фиксирует, с какого устройства и через какой источник (прямой переход, поисковая выдача, Яндекс.Директ, Avito или другие каналы) пользователь попал на ваш сайт. Все эти данные собираются в едином интерфейсе, что значительно упрощает их анализ.

Roistat также умеет объединять визиты одного клиента, даже если он заходит на сайт с разных устройств. Например, если пользователь оставляет заявку или совершает заказ, система связывает эти действия в одну цепочку на основе совпадающих данных, таких как номер телефона или email. Это позволяет отследить весь путь клиента и точно определить, какой рекламный канал принёс результат.

<figure><img src="../../../.gitbook/assets/image (310).png" alt=""><figcaption></figcaption></figure>

Для наглядного представления данных в Roistat предусмотрена **диаграмма**, которую можно настроить с помощью параметров, расположенных справа.

С помощью кнопки **"Добавить активность"** вы можете отмечать на диаграмме ключевые события, например, запуск новой рекламной кампании. Такие метки помогают понять, как изменения в рекламе влияют на активность на сайте: отследить всплески или спады, оценить эффективность кампании и принять обоснованные решения для дальнейших действий.

<figure><img src="../../../.gitbook/assets/image (3875).png" alt=""><figcaption></figcaption></figure>
