# Выписки по расписанию - СберБанк



Подключите "Выписки по расписанию" на [странице СберБанка](https://www.sberbank.ru/ru/s_m_business/nbs/schedule). Для этого перейдите по ссылке и нажмите "Подключить"

<figure><img src="../../../.gitbook/assets/Screenshot_1 (2).png" alt=""><figcaption></figcaption></figure>

После авторизации на странице СберБанка нужно указать почту, которая написана в интеграции. Для этого откройте Админ-панель → Настройки → Интеграции → Финансы → СберБанк.

<figure><img src="../../../.gitbook/assets/Screenshot_2 (17).png" alt=""><figcaption></figcaption></figure>

В окне интеграции СберБанка будет указана почта, которую нужно будет ввести на сайте СберБанка.

<figure><img src="../../../.gitbook/assets/Screenshot_3 (4).png" alt=""><figcaption></figcaption></figure>

В СберБанке Формат выписки необходимо выбрать 1С, кодировка Win. Также вы можете указать дополнительную почту.

<figure><img src="../../../.gitbook/assets/Screenshot_4 (27).png" alt=""><figcaption></figcaption></figure>

В интеграции нажмите на "Добавить компанию", во всплывающем окне выберите реквизиты нужной компании и сохраните изменения.

<figure><img src="../../../.gitbook/assets/Screenshot_5 (23).png" alt=""><figcaption></figcaption></figure>
