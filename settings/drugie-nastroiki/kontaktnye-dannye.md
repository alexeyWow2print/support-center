# Контактные данные

## Контактные данные

На данной странице можно заполнить основную контактную информацию о компании:

* _Контактный телефон типографии_ — отображается в шапке сайта и в [футере](https://support.wow2print.com/content/untitled/menyu/futer) в правой колонке;
* _Подпись телефона_ — отображается в шапке сайта рядом с контактным номером типографии;

{% hint style="danger" %}
Подпись и номер телефона в шапке сайта могут отличаться от данных, введенных в этом разделе. Это может происходить из-за настроек [филиала](https://support.wow2print.com/settings/dostavka/filialy#filialy).
{% endhint %}

* _Наименование, отображаемое на сайте_ — отображается в [футере](https://support.wow2print.com/content/untitled/menyu/futer) сайта в левой колонке;
* _Наименование, отображаемое в письмах_ — отображается в письме в графе "Имя отправителя";
* _E-mail типографии_ — отображается в [футере](https://support.wow2print.com/content/untitled/menyu/futer) сайта в правой колонке;
* _Год начала работы_ — отображается в [футере](https://support.wow2print.com/content/untitled/menyu/futer) сайта в левой колонке;
* _Адрес типографии_ — отображается в [футере](https://support.wow2print.com/content/untitled/menyu/futer) сайта в правой колонке.

## Социальные сети

Также имеется возможность указать ссылки на социальные сети:

* Вконтакте;
* Facebook;
* Instagram;
* Telegram;
* Max;

{% hint style="warning" %}
Ссылка на социальную сеть имеет вид - "https://vk.me/tippocloudservice", в настройках соц.сетей ее необходимо указать без адреса сайта: ~~https://vk.com/~~tippocloudservice.&#x20;
{% endhint %}

* Whatsapp;
* Viber.

Для мессенджеров достаточно указать номер телефона.

{% hint style="info" %}
Иконка социальной сети отображается в [футере](https://support.wow2print.com/content/untitled/menyu/futer) только в том случае, если соответствующее поле заполнено в разделе "Контактные данные".
{% endhint %}
