# Яндекс

Яндекс → [https://oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new)

Необходимо перейти по указанной ссылке и заполнить необходимые поля:

* _Название сервиса_ — название приложения, которое будет отображаться для клиента;
* _Иконка_ — при необходимости, можете загрузить иконку приложения;
* _Платформы_ — выбираем "Веб-сервисы";
* _Redirect URI_ — в этом поле необходимо указать адрес сайта в виде _https://**mydomain.ru**/social/login/service/yandex._ Замените _**mydomain.ru**_ на домен своего сайта;
* _Доступ к данным_ — выбираем все имеющиеся доступы.

* _Почта для связи - &#x443;_&#x43A;ажите почту компании или свою, на которую будут приходить оповещения об изменениях во внешней авторизации.

И нажмите Сохранить.

<figure><img src="../../../.gitbook/assets/Screenshot_2 (20).png" alt=""><figcaption></figcaption></figure>



После создания приложения, необходимо скопировать следующие данные:

* _Client Id_ — это ID;
* _Client Secret_ — это Пароль.

Их необходимо разместить в соответствующих полях в админ-панели сайта:

> Админ-панель → Настройки → Другие настройки → Авторизация через соцсети

Сохраните изменения.



Авторизация через соцсети работает следующим образом:

{% hint style="info" %}
При попытке авторизоваться ПЕРВЫЙ раз, происходит регистрация клиента на сайте — своеобразная привязка учетной записи социальной сети с личным кабинетом на сайте. Все последующие попытки авторизоваться через соцсети, будут происходить буквально в один клик.
{% endhint %}
