# Настройка FTP-сервера

В Настройки → Другие настройки → Настройка FTP-сервера вы можете настроить автоматическую передачу файлов из системы на свой сервер.

![](<../../.gitbook/assets/image (3206).png>)

Для этого, заполните поля:

**Сервер**

Укажите сервер, на который будет осуществляться отправка файлов

**Пользователь и Пароль**

Укажите пользователя (логин) и пароль от сервера, куда планируется отправлять файлы

**Директория загрузки**

Директория загрузки - папка, куда будут загружены макеты. Если директория не существует, то она будет создана. В названии директории нужно указать шаблоны {orderId} и {productId}, которые будут заменены на номер заказа и номер продукта соответственно.

**Использовать пассивный режим передачи файлов**

Синхронизируйте режимы передачи файлов своего FTP- сервера и системы TCS. Для корректной передачи файлов, режимы должны совпадать. В случае, если не выбрано "Использовать пассивный режим передачи файлов" по умолчанию режим считается активным.



Выберите опции: **Архивировать макеты** и\или **Загружать исходные макеты.** При выборе **Загружать исходные макеты,** на сервер будут поступать: 1.  макет, обработанный системой и 2) исходник, загруженный клиентом.

Условия загрузки макето&#x432;**: Макет принят**/**Заказ оплачен или есть гарантия оплаты заказа.** В случае выбора услови&#x44F;**,** макет будет выгружаться только при их выполнении. Если условия не выбраны, на FTP-сервер будут передаваться файлы всех макетов.
