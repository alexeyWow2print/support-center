# Название макетов

Вы можете создать удобные вам шаблоны названия макетов для скачивания с указанием тех параметров, которые вы добавите в шаблон.



### Создание шаблона

Чтобы добавить новый шаблона названия в Настройки → Другие настройки→ Работа с макетом → Названия макетов → нажмите кнопку "Добавить"

![](<../../../.gitbook/assets/image (546).png>)



В открывшейся форме:

* Задайте название шаблону
* Сформируйте шаблон, перетягивая в соответствующее поле необходимые элементы и располагая их в той последовательности, в которой вы хотите видеть их в названии макета.

![](<../../../.gitbook/assets/image (1079).png>)

* Выберите продукцию, к названиям макетов которой необходимо будет применять данный шаблон

![](<../../../.gitbook/assets/image (2380).png>)

&#x20;В названии скачанного макета будут указываться те параметры, которые вы добавите в шаблон.

\
Для разной продукции можно создавать свои отдельные шаблоны, отображая в названии актуальные для того или иного продукта параметры.



### Удаление шаблона

Чтобы удалить шаблона нажмите на кнопку "Удалить" напротив нужного шаблона.

![](<../../../.gitbook/assets/image (766).png>)
