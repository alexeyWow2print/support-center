# Шаблоны email-писем

## Общие положения

На сайте предусмотрена автоматическая отправка электронных писем. Письма отправляются по определенному шаблону, как клиенту, так и пользователю (менеджеру) сайта.\
Отправка писем настраивается в [настройках почтового сервера](nastroika-pochtovogo-servera.md).\
В данном разделе можно настроить шаблон (текст) писем и отключить отправку некоторых из них.

{% hint style="warning" %}
Письма отправляются только по заранее заданному шаблону и только автоматически.\
Нельзя написать и вручную отправить письмо из админ-панели сайта.
{% endhint %}

### Редактирование шаблона

Любой из представленных шаблонов можно редактировать, для этого нажмите на его название.

![](<../../.gitbook/assets/image (3367).png>)



В левой части располагается:

* **Title**\
  Будет отображаться как тема письма.<br>
* **Body**\
  Текст письма. Весь текст воспринимается как html-код.<br>

В правой части располагаются теги, которые можно использовать в тексте письма:

* **{LogoUrl}**\
  Логотип сайта.<br>
* **{CompanyName}**\
  Название компании — берется из [контактных данных](kontaktnye-dannye.md) из поля «Наименование, отображаемое в письмах».<br>
* **{CompanyPhone}**\
  Телефон компании — берется из [контактных данных](kontaktnye-dannye.md) из поля «Контактный телефон типографии».<br>
*   **{CompanyEmail}**\
    Email компании — берется из [контактных данных](kontaktnye-dannye.md) из поля «Email типографии».


* **{CabinetUrl}**\
  Личный кабинет клиента — ссылка для входа в ЛК.<br>
* **{OrderInfo}**\
  Информация о заказе — таблица с указанием всех позиций заказа.<br>
* **{ManagerName}**\
  Имя менеджера.<br>
* **{ManagerPhone}**\
  Телефон менеджера.<br>
* **{ManagerEmail}**\
  Email менеджера.<br>
* **{ManagerInfo}**\
  Информация о менеджере — небольшой прямоугольный блок, в котором указаны: Фамилия, Имя, Email и Телефон менеджера.<br>
* **{ClientFirstName}**\
  Имя клиента.<br>
* **{ClientLastName}**\
  Фамилия клиента.<br>
* **{ClientMiddleName}**\
  Отчество клиента.<br>
* **{ClientEmail}**\
  Почта клиента.<br>
* **{ClientPhone}**\
  Номер клиента.<br>
* **{ClientPayer}**\
  Плательщик.<br>
* **{CashbackTotalCount}**\
  Общее количество бонусов — текущее количество бонусов клиента.<br>
* **{CashbackOrder}**\
  Бонусы за текущий заказ.<br>
* **{CreditedCashbackDate}**\
  Дата зачисления бонусов.<br>
* **{ClientNotLoadLayouts}**\
  Не загруженные макеты клиента — таблица с перечислением всех позиций, на которые необходимо загрузить макет.<br>
* **{CashbackCreditedCount}**\
  Количество зачисленных бонусов.<br>
* **{InternalBalanceCount}**\
  Сумма внутреннего баланса.
