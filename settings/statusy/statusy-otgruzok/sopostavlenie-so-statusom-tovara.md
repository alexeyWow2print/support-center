# Сопоставление статусов отгрузок со статусами товаров

Статусы отгрузок необходимо сопоставить со статусами товаров — при изменении статуса отгрузки будет автоматически меняться статус товаров.

![](<../../../.gitbook/assets/image (2711).png>)



Пример: Статус отгрузки меняем на "Отгружен" → Товары из данной отгрузки автоматически меняют статус на "Завершен/Выдан".

Нет необходимости менять статусы отправленных товаров из данной отгрузки вручную.
