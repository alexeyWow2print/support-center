# Создание статусов отгрузок

Для того, чтобы настроить статусы отгрузок вам необходимо отредактировать название имеющихся статусов или добавить недостающие с помощью кнопки "Добавить статус" в  Настройки → Статусы → Статусы отгрузок.

![](<../../../.gitbook/assets/image (2545).png>)

Заполните поле _Название,_ нажмите "Сохранить" и Статус появится в общем списке

![](<../../../.gitbook/assets/image (3100).png>)



{% hint style="info" %}
Первый и последний статус стоят по умолчанию и не подлежат удалению. Для Вашего удобства названия статусов можно переименовать.
{% endhint %}





**Редактирование статусов**

Чтобы отредактировать название существующего статуса, введите новое название в соответствующее поле и нажмите "Сохранить"

![](<../../../.gitbook/assets/image (2518).png>)





**Удаление статусов**

Статусы, кроме "Новый" и "Завершенный", вы можете удалить. Для этого нажмите на иконку "корзина" напротив нужного статуса и подтвердите удаление.

![](<../../../.gitbook/assets/image (1651).png>)
