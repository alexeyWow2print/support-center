# Цветовая сигнализация статусов и сроков исполнения

Назначение своего цвета каждому статусу отгрузки позволит легко ориентироваться в общем списке отгрузок.&#x20;

Чтобы выбрать свой цвет для каждого статуса, нажмите на иконку слева от названия статуса.

![](<../../../.gitbook/assets/image (1130).png>)



Справа от названия статуса укажите время, которое отведено на выполнение каждого статуса.

![](<../../../.gitbook/assets/image (2656).png>)



Цветовая сигнализация статусов отгрузки изменяется в зависимости от процента прошедшего времени от указанного для выполнения статуса. Принцип действия аналогичен [принципу действия цветовой сигнализации заказов.](../statusy-zakazov/cvetovaya-signalizaciya-statusov.md#princip-cvetovoi-signalizacii)
