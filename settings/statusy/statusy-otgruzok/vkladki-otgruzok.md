# Вкладки Отгрузок

Для удобства работы и возможности отслеживать отгрузку по степени готовности, можно создать несколько вкладок.

![](<../../../.gitbook/assets/image (791).png>)



Чтобы создать вкладку нажмите кнопку "Добавить вкладку".

![](<../../../.gitbook/assets/image (1097).png>)

Заполните поле _Название_,  нажмите "Сохранить" и вкладка появится в общем списке вкладок

![](<../../../.gitbook/assets/image (1072).png>)



В колонке "Вкладка" с помощью выпадающего списка необходимо настроить соотношение статуса отгрузки с вкладкой, на которой будет отображаться данный статус.

![](<../../../.gitbook/assets/image (2495).png>)
