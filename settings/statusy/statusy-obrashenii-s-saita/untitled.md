# Удаление статусов обращений с сайта и вкладок

**Удаление статусов**

Чтобы удалить статус нажмите на кнопку "Удалить" напротив нужного статуса.

![](<../../../.gitbook/assets/image (2465).png>)

Обращаем ваше внимание, что статусы "Новый" и "Завершен" созданы по умолчанию, без возможности их удаления.



**Удаление вкладок**

Чтобы удалить вкладку, нажмите кнопку "Удалить" напротив нужной вкладки.

![](<../../../.gitbook/assets/image (3115).png>)

Ограничения по удалению вкладок нет, можно удалить любую созданную вкладку.
