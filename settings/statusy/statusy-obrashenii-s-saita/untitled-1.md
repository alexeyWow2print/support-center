# Сопоставление статусов обращения с вкладками

### Вкладки&#x20;

Настройка вкладок позволит упорядочить обращения с сайта в зависимости от стадии выполнения: переходя от статуса к статусу обращения также будут перемещаться из одной вкладки в другую.

Чтобы добавить вкладку нажмите кнопку "Добавить вкладку".

![](<../../../.gitbook/assets/image (2451).png>)

Заполните поле _Название_, нажмите "Сохранить" и новая вкладка появится в общем списке вкладок.

![](<../../../.gitbook/assets/image (1107).png>)



### Сопоставление статусов и вкладок

В колонке "Вкладка" с помощью выпадающего списка необходимо настроить соотношение статуса обращения с вкладкой, на которой будет отображаться данный статус.

![](<../../../.gitbook/assets/image (3186).png>)



### Сопоставление статусов обращений со статусами заказов.

Статусы обращения можно сопоставить со статусами заказов.&#x20;

![](<../../../.gitbook/assets/image (990).png>)

Если на основе обращения создан заказ, то при перемещении заказа по статусам обращение тоже будет перемещаться по статусам, назначенным в сопоставлении.
