# Создание статусов обращений

Для того, чтобы настроить статусы обращений с сайта вам необходимо отредактировать название имеющихся статусов или добавить недостающие с помощью кнопки "Добавить статус" в  Настройки → Статусы → Статусы  обращений с сайта.&#x20;



![](<../../../.gitbook/assets/image (2695).png>)



Заполните поле _Название_, нажмите "Сохранить" и Статус появится в общем списке

![](<../../../.gitbook/assets/image (2500).png>)



{% hint style="info" %}
Первый и последний статус стоят по умолчанию и не подлежат удалению. Для Вашего удобства названия статусов можно переименовать.
{% endhint %}





**Редактирование статусов**

Чтобы отредактировать название существующего статуса, введите новое название в соответствующее поле и нажмите "Сохранить"

![](<../../../.gitbook/assets/image (2649).png>)
