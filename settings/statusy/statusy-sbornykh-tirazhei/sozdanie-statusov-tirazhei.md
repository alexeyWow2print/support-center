# Создание статусов тиражей

### **Добавление нового статуса**

Для того, чтобы настроить статусы тиражей, вам необходимо отредактировать название имеющихся статусов или добавить недостающие с помощью кнопки "Добавить статус".

![](<../../../.gitbook/assets/image (3762).png>)



{% hint style="info" %}
Первый и последний статус стоят по умолчанию и не подлежат удалению. Для Вашего удобства названия статусов можно переименовать.
{% endhint %}



Заполните поле _Название_, нажмите "Сохранить" и статус появится в общем списке статусов

![](<../../../.gitbook/assets/image (2926).png>)





### **Редактирование статуса**

Чтобы отредактировать название существующего статуса, введите новое название в соответствующее поле.

![](<../../../.gitbook/assets/image (3417).png>)





### Удаление статуса

Чтобы удалить статус нажмите на кнопку "Удалить" напротив статуса.

![](<../../../.gitbook/assets/image (3585).png>)
