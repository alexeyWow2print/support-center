# Вкладки тиражей

Для удобства работы и возможности отслеживать тираж по степени готовности, можно создать несколько вкладок.

![](<../../../.gitbook/assets/image (2714).png>)



Чтобы создать новую вкладку нажмите "Добавить вкладку" и заполните строку _Название_, после этого она появится в списке вкладок

![](<../../../.gitbook/assets/image (1087).png>)



Сопоставление "Статуса" и "Вкладки"

Статусы тиражей необходимо сопоставить с Вкладками. Для этого в колонке "Вкладка" с помощью выпадающего списка необходимо настроить соотношение статуса тиража с вкладкой, на которой будет отображаться данный статус.

![](<../../../.gitbook/assets/image (2522).png>)
