# Сопоставление статусов тиражей со статусами товаров

Статусы тиражей необходимо сопоставить со статусами товаров — при изменении статуса тиража будет автоматически меняться статус товаров.

![](<../../../.gitbook/assets/image (2818).png>)

Пример: Тираж отпечатан → Меняем статус тиража на "Готов" → Товары данного тиража автоматически меняют статус на "Проверка/Комплектация".

Нет необходимости менять статусы отпечатанных товаров из данного тиража вручную.
