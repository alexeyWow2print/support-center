# Создание статусов

**Создание статусов**

Для того, чтобы настроить статусы заказов и товаров вам необходимо отредактировать название имеющихся статусов или добавить недостающие с помощью кнопки "Добавить статус"&#x20;



![](<../../../.gitbook/assets/image (2485).png>)



заполнить _Название_ → "Сохранить" и статус появится в общем списке.

![](<../../../.gitbook/assets/image (1667).png>)



{% hint style="info" %}
Первый и последний статус стоят по умолчанию и не подлежат удалению. Для Вашего удобства названия статусов можно переименовать
{% endhint %}

\
Чтобы отредактировать название существующего статуса, введите новое _название_ в  соответствующее поле.

![](<../../../.gitbook/assets/image (2931).png>)



Чтобы поменять местами статусы достаточно перетянуть строку в нужное место с помощью иконки, которая находится слева от названия статуса.

![](<../../../.gitbook/assets/image (2601).png>)

Цветовая сигнализация изменяется в зависимости от процента прошедшего времени от указанного для выполнения статуса.
