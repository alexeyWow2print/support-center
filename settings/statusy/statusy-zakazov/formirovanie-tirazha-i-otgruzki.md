# Формирование тиража и отгрузки

**Формирование тиража**

Для формирования тиражей, отметьте статусы, на которых будет доступно формирование тиража

![](<../../../.gitbook/assets/image (2467).png>)



&#x20;**Требования формирования тиража:**

* макет должен быть загружен и принят
* во всех товарах тиража должен использоваться один материал (например, Мелованная бумага, плотность 115 гр.)
* сформировать тираж можно только из товаров.
* заказ должен быть оплачен или стоять отметка о гарантии оплаты.



**Формирование отгрузки**

Для формирование отгрузок отметьте статусы, на которых доступно формирование отгрузки:

![](<../../../.gitbook/assets/image (2703).png>)

&#x20;

**Требования формирования отгрузки:**

* заказ должен быть оплачен (или проставлена гарантия оплаты)
* объединить несколько товаров в одну отгрузку можно заказы только одного клиента.
* сформировать отгрузку можно только из товаров.<br>
