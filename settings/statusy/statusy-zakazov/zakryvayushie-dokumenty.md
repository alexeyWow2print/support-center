# Закрывающие документы

В статусах заказов вы можете настроить список статусов при которых формируются закрывающие документы.

![](<../../../.gitbook/assets/image (2456).png>)



При подключении модуля Отгрузки, после создания отгрузки, все отгрузки отображаются во вкладке "Новый" в&#x20;

> Магазин → Отгрузки

Когда отгрузка получает статус на котом формируются закрывающие документы, данные документы можно скачать в&#x20;

> Магазин → Бухгалтерия → Реализация

Дату закрывающих документов вы можете корректировать.
