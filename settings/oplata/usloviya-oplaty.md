# Условия оплаты

Добавьте условия оплаты, которые в дальнейшем можно будет назначать отдельным клиентам (если данный доп.модуль подключен) через кнопку "Добавить в правом углу".

![](<../../.gitbook/assets/image (1082).png>)



По умолчанию каждому клиенту сайта устанавливается условие 100% предоплаты.

![](<../../.gitbook/assets/image (1126).png>)



Редактировать условия оплаты клиентам можно в профиле клиента Магазин→ Контрагенты → Профиль клиента

![](<../../.gitbook/assets/image (2228).png>)



Выберите тип оплаты (предоплата или постоплата). В случае предоплаты установите процент.

![](<../../.gitbook/assets/image (1070).png>)



При необходимости укажите лимит задолженности.

![](<../../.gitbook/assets/image (781).png>)

После превышения максимальной суммы задолженности клиент автоматически переводится на предоплату 100%.&#x20;

После погашения задолженности вновь применяется установленное для него условие оплаты.
