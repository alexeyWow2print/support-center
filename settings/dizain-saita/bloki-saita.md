# Блоки сайта

## Общие положения

Шапка и футер отображаются на всех страницах сайта и имеют стандартный вид. В админ-панели имеется возможность изменить стандартный вид шапки и футера с помощью верстки html-кода.

![](<../../.gitbook/assets/image (1056).png>)



Статус (ползунок) напротив каждого блока отвечает за вариант отображения.

* **Включен**\
  Применяется индивидуальная верстка, код которой располагается внутри данного блока.<br>
* **Выключен**\
  Применяется стандартный (по умолчанию) вид блока.





### Редактор блока

При нажатии на один из блоков, откроется страница с инструментом редактирования.

![](<../../.gitbook/assets/image (870).png>)



В левой части располагается:

* **Режим тестирования**\
  При активации данного режима, индивидуальную верстку на сайте будут видеть только авторизованные пользователи. Обычные клиенты сайта будут видеть стандартную верстку.\
  Если вы только начинаете вносить изменения в данный блок, то воспользуйтесь режимом тестирования.

{% hint style="danger" %}
Режим тестирования работает только при статусе блока «Выключен».
{% endhint %}

* **html**\
  Текстовый редактор, в который необходимо располагать индивидуальную верстку.<br>

В правой части располагаются теги, которые можно использовать в коде верстки блока:

* **{LogoUrl}**\
  Логотип сайта.<br>
* **{LogoAlt}**\
  Alt для логотипа — берется из настроек логотипа.<br>
* **{CityLocationWidget}**\
  Виджет для выбора города.<br>
* **{CabinetProfileName}**\
  Имя авторизованного клиента.<br>
* **{CabinetProfileUrl}**\
  Ссылка на профиль.<br>
* **{CabinetOrdersUrl}**\
  Ссылка на заказы.<br>
* **{CabinetExitUrl}**\
  Ссылка на выход.<br>
* **{CartUrl}**\
  Ссылка на корзину.<br>
* **{CompanyName}**\
  Название компании — берется из [контактных данных](../drugie-nastroiki/kontaktnye-dannye.md) из поля «Наименование, отображаемое на сайте».<br>
* **{CompanyPhone}**\
  Телефон компании — берется из [контактных данных](../drugie-nastroiki/kontaktnye-dannye.md) из поля «Контактный телефон типографии».<br>
*   **{CompanyAddress}**\
    Адрес компании — берется из [контактных данных](../drugie-nastroiki/kontaktnye-dannye.md) из поля «Адрес типографии».


*   **{CompanyEmail}**\
    Email компании — берется из [контактных данных](../drugie-nastroiki/kontaktnye-dannye.md) из поля «Email типографии».


* **{WidgetMenu1}**\
  Меню «Верхнее меню».<br>
* **{WidgetMenu2}**\
  Меню «Меню в футере (левая колонка)».<br>
* **{WidgetMenu3}**\
  Меню «Меню в футере (правая колонка)».
