# Менеджер шрифтов

В этом разделе вы можете загружать собственные шрифты, которые будут использоваться для оформления вашего сайта. Настройка внешнего вида сайта производится в разделе [**«Типографическая система»**](tipograficheskaya-sistema.md).

<figure><img src="../../.gitbook/assets/image (4078).png" alt=""><figcaption></figcaption></figure>







### Обзор менеджера шрифтов

На главной странице раздела представлен список доступных шрифтов. По умолчанию здесь уже есть **7 основных шрифтов** платформы.

Список содержит две колонки:

* **«Название»** — здесь указано имя шрифта.
* **«Визуализация»** — пример написания: название шрифта + фраза _«Aa Bb Cc 123»_.

{% hint style="info" %}
**Важно:** Вы можете добавить не более **5 собственных шрифтов**.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (4079).png" alt=""><figcaption></figcaption></figure>







### Как добавить новый шрифт

1. Нажмите кнопку **«Добавить шрифт»** в правом верхнем углу экрана.
2. Вы перейдёте на страницу загрузки шрифта.
3. В поле **«Название шрифта»** обязательно укажите имя. Без него сохранение будет невозможно.
4. Ниже расположены 4 блока для загрузки начертаний:
   * **Regular (обязательно)**
   * Medium
   * SemiBold
   * Bold

<figure><img src="../../.gitbook/assets/image (4080).png" alt=""><figcaption></figcaption></figure>







#### Требования к загрузке файлов

{% hint style="warning" %}
* **Форматы:** `.ttf`, `.otf`, `.woff`, `.woff2`
* **Обязательное условие:** начертание **Regular** должно быть загружено всегда.
{% endhint %}



**Как загрузить файл:**

1. Нажмите кнопку **«Выберите файл»** в нужном начертании.
2. Найдите файл шрифта на вашем компьютере.
3. После успешной загрузки справа от кнопки появится название загруженного файла.

<figure><img src="../../.gitbook/assets/image (4081).png" alt=""><figcaption></figcaption></figure>







#### Управление загруженными файлами

* После сохранения изменений название загруженного файла будет отображаться под строкой соответствующего начертания.
* **Заменить файл** можно в любой момент — просто загрузите новый файл в том же начертании.
* **Удалить стиль** (начертание): после сохранения у всех стилей, кроме **Regular**, появляется кнопка **«Удалить стиль»**.

<figure><img src="../../.gitbook/assets/image (4082).png" alt=""><figcaption></figcaption></figure>







### Удаление собственного шрифта

Любой добавленный вами шрифт (кроме стандартных семи) можно удалить.

1. Найдите нужный шрифт в списке.
2. Справа в строке шрифта нажмите кнопку с иконкой **корзины**.
3. Подтвердите действие.

Шрифт будет удалён из списка.
