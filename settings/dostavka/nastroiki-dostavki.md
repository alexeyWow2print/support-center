# Настройки доставки

## Общие положения

В данном разделе можно увидеть следующие параметры

![](<../../.gitbook/assets/image (863).png>)



* **Округление стоимости доставки**\
  Стоимость доставки по интеграции может подтягиваться с копейками, данный параметр позволяет округлить стоимость до целых значений.<br>
* **Страна по умолчанию**\
  Выберите страну, в которой расположена типография<br>
* **Параметр для калькуляции доставки**\
  В некоторых европейских странах есть возможность указывать вместо города почтовый индекс, если рассматривать Россию и страны СНГ, то доставка преимущественно считается именно на основании города<br>
* **Доставка по странам**\
  В данном списке необходимо отметить те страны, в которые производится доставка продукции.
